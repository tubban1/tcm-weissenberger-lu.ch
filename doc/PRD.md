# TCM Weissenberger 产品需求文档（PRD）

## 一、项目目标

### 1.1 项目概述
TCM Weissenberger 是一家位于瑞士的中医诊所官方网站，旨在为瑞士患者提供专业、可信的中医诊疗服务信息，并支持在线预约和咨询功能。网站采用瑞士本地化设计，强调科学严谨与自然疗愈的结合，符合瑞士医疗市场标准和患者偏好。

### 1.2 项目目标
- **品牌展示**：建立专业、可信的瑞士中医诊所品牌形象，强调"自然疗愈 + 科学严谨"的定位
- **服务介绍**：清晰展示诊所提供的各类中医服务项目和诊疗范围，以症状为导向提供实用信息
- **在线预约**：提供极简的一键预约功能，支持 Google Calendar 同步，提升患者体验
- **信息传达**：向患者传递诊所信息、联系方式、营业时间、保险报销等关键信息
- **用户体验**：采用瑞士本地化审美设计，简洁清爽，易于导航，适配移动端
- **差异化定位**：突出"瑞士首个基于 ICD-11 的智能中医病历系统"等独特技术优势
- **多语言支持**：提供德语、英语、中文三种语言，德语优先，满足瑞士多语言环境需求

### 1.3 设计原则
- **瑞士本地化审美**：简约、清爽、可信、结构明确，符合瑞士医疗机构视觉风格
- **专业性**：体现中医诊所的专业性和可信度，强调认证资质和科学依据
- **易用性**：导航清晰，功能易于使用，适配移动端，避免复杂流程
- **科学化呈现**：结合现代医学研究，避免过度"东方神秘感"，强调"现代整合、科学导向"
- **患者中心**：以患者需求为导向，提供症状导向的内容和便捷的预约体验

## 二、系统用户与角色

### 2.1 主要用户群体

#### 2.1.1 患者/潜在患者
- **需求**：
  - 了解诊所服务项目和诊疗能力
  - 查看诊所信息和联系方式
  - 在线预约就诊时间（无需注册账号）
  - 进行健康咨询
  - 了解保险报销方式和认证信息
  - 查看症状导向的治疗方案
  - 了解交通方式和诊所位置
- **特点**：
  - 可能不熟悉中医服务，需要科学化的解释
  - 需要清晰的信息展示和引导
  - 非常重视诊所的专业性、可信度和认证资质
  - 偏好简洁、透明的信息呈现
  - 重视时间管理和预约便利性
  - 需要德语、英语或中文的语言支持

#### 2.1.2 诊所工作人员（管理员）
- **需求**：
  - 查看和管理预约信息
  - 更新服务内容和诊所信息
  - 查看患者咨询和留言
- **特点**：
  - 需要简单的后台管理功能
  - 通常为非技术人员

#### 3.1.3 服务项目（Services）
**功能描述**：
- 详细展示诊所提供的所有中医服务项目
- 每个服务包含：服务名称、描述、主要功效、适用症状
- **科学化呈现**：结合现代医学研究，引用相关科学文献
- 避免过度传统叙事，强调现代医学与中医的结合

**服务项目列表**（德语/英语）：
1. **Akupunktur（针灸治疗）**：
   - 描述：传统针灸疗法，缓解疼痛，调理身体
   - 科学依据："Studien zeigen, dass Akupunktur die Schmerzverarbeitung im zentralen Nervensystem moduliert..."
   - 主要功效：缓解疼痛、调节神经系统、改善循环
   - 适用症状：Rückenschmerzen, Migräne, Stress, Schlafprobleme

2. **Chinesische Kräutermedizin（中药调理）**：
   - 描述：个性化中药配方，调理体质，增强免疫力
   - 主要功效：调理体质、增强免疫力、平衡阴阳
   - 适用症状：Verdauungsprobleme, chronische Müdigkeit, Immunschwäche

3. **Tuina-Massage（推拿按摩）**：
   - 描述：专业推拿手法，舒缓筋骨，放松身心
   - 主要功效：舒缓筋骨、放松身心、改善循环
   - 适用症状：Muskelverspannungen, Rückenschmerzen, Stress

4. **Schröpfen（拔罐疗法）**：
   - 描述：疏通经络，活血化瘀，缓解肌肉酸痛
   - 主要功效：疏通经络、活血化瘀、缓解肌肉酸痛
   - 适用症状：Muskelschmerzen, Verspannungen, Erkältung

5. **Moxibustion（艾灸治疗）**：
   - 描述：温经散寒，补气养血，增强体质
   - 主要功效：温经散寒、补气养血、增强体质
   - 适用症状：Kältegefühl, chronische Müdigkeit, Menstruationsbeschwerden

6. **TCM-Beratung（中医咨询）**：
   - 描述：健康评估，养生指导，生活方式建议
   - 主要功效：健康评估、养生指导、生活方式建议
   - 适用症状：Prävention, allgemeine Gesundheitsberatung

**交互要求**：
- 表单验证
- 实时显示可用时间段
- 提交后显示确认信息
- 自动发送确认邮件（包含 Google Calendar / Apple Calendar 链接）

## 四、技术实现

### 4.1 技术栈
- **框架**：Next.js 14（App Router）
- **语言**：TypeScript
- **样式**：Tailwind CSS
- **字体**：Helvetica / Lato（瑞士风格），Inter 作为备用
- **国际化（i18n）**：next-intl 或 next-i18next
  - 支持语言：德语（de，默认）、英语（en）、中文（zh-CN）
  - 路由结构：`/[locale]/...` 或使用子域名/路径前缀
- **日历集成**：Google Calendar API（预约系统）
- **邮件服务**：SendGrid / Resend（预约确认邮件）

### 4.3 设计规范

#### 4.3.1 色彩方案（瑞士本地化审美）
- **主色调**：自然疗愈风格
  - Primary Green（松木绿）：#2d5016 / #4a7c2a / #6b9f3d（体现自然、健康）
  - Accent（暖米色）：#f5f1e8 / #e8e0d1（温暖、舒适）
- **中性色**：瑞士简约风格
  - 文本：gray-900, gray-700, gray-600
  - 背景：white, gray-50, gray-100
  - 边框：gray-200, gray-300
- **强调色**：
  - 成功/确认：green-600
  - 警告：yellow-500
  - 错误：red-500
- **避免**：过度使用红色（传统中医色），避免"东方神秘感"配色

#### 4.3.2 字体（瑞士视觉风格）
- **主字体**：Helvetica / Lato（瑞士常用字体）
  - 标题：Helvetica Neue / Lato Bold
  - 正文：Helvetica / Lato Regular
- **备用字体**：Inter（Google Fonts，与 Helvetica 相似）
- **字体大小**：
  - h1: 4xl (2.25rem)
  - h2: 3xl (1.875rem)
  - h3: 2xl (1.5rem)
  - 正文：base (1rem)
  - 小字：sm (0.875rem)

#### 4.3.3 排版与布局
- **大量留白**：符合瑞士简约设计，页面留白充足
- **明确模块分区**：About / Behandlung / Preise / Termin 等模块清晰分隔
- **层次分明**：标题、正文、说明文字层次清晰
- **间距**：使用 Tailwind 标准间距单位，保持一致性

#### 4.3.4 图片风格
- **高质量瑞士本地环境**：阿尔卑斯、草木、诊所实拍
- **避免**：过度"东方神秘感"的传统图案、壁画
- **风格**：现代、专业、自然、可信

#### 4.3.5 响应式设计
- 移动端优先（Mobile First）
- 断点：sm (640px), md (768px), lg (1024px), xl (1280px)
- 导航栏移动端使用汉堡菜单
- 确保所有功能在移动端可用

## 五、后续扩展功能（非 MVP）

### 5.1 AI 功能集成（差异化特色）
#### 5.1.1 AI 生成的双语病历（ICD-11 标准化）
- **功能**：自动生成符合 ICD-11 标准的双语病历（德语/英语）
- **优势**：瑞士中医诊所几乎没有此功能 → 差异化极强
- **技术**：集成 AI 模型，基于诊疗记录生成标准化病历
- **输出**：结构化病历文档，符合 WHO-ICD-11 分类标准

#### 5.1.2 治疗前后可视化分析（周期图表）
- **功能**：展示治疗前后的数据对比和趋势分析
- **图表类型**：
  - Migraine Frequenz (Vorher/Nachher) - 偏头痛频率
  - Schlafqualität - 睡眠质量
  - Stresslevel - 压力水平
  - Schmerzintensität - 疼痛强度
- **可视化**：使用 Chart.js 或 Recharts 生成交互式图表
- **价值**：帮助患者直观了解治疗效果

#### 5.1.3 在线健康问卷与初诊准备
- **功能**：患者预约前填写健康问卷
- **内容**：症状描述、病史、生活习惯等
- **价值**：减轻医师工作量，让患者更安心
- **输出**：生成初诊准备报告，医师可提前了解患者情况

## 七、注意事项

1. **内容准确性**：
   - 确保所有医疗相关信息准确、合规
   - 科学依据和研究引用需真实可靠
   - 避免夸大治疗效果

2. **隐私保护**：
   - 患者信息需要符合瑞士数据保护法规（DSG）
   - 预约信息加密存储
   - 符合 GDPR 要求（如适用）

3. **多语言内容质量**：
   - 德语内容优先，确保专业性和准确性
   - 英语内容需符合瑞士英语习惯
   - 中文内容保持专业术语一致性
   - 所有翻译需由专业翻译或母语者审核

4. **瑞士医疗法规合规**：
   - 确保保险报销信息准确
   - ASCA / EMR 认证信息需真实有效
   - 价格信息需透明、准确
   - 避免医疗广告违规

5. **可访问性**：
   - 遵循 Web 可访问性标准（WCAG 2.1）
   - 确保多语言切换无障碍
   - 移动端体验优化

6. **加载性能**：
   - 确保图片和资源优化加载
   - 多语言资源按需加载
   - 使用 Next.js Image 优化

7. **备份机制**：
   - 定期备份网站数据和配置
   - 多语言内容备份
   - 预约数据备份

8. **品牌一致性**：
   - 保持瑞士本地化审美风格
   - 避免过度"东方神秘感"
   - 强调科学严谨与自然疗愈结合
