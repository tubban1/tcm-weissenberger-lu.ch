# TCM Weissenberger 产品需求文档（PRD）

## 一、项目目标

### 1.1 项目概述
TCM Weissenberger 是一家中医诊所的官方网站，旨在为患者提供专业的中医诊疗服务信息，并支持在线预约和咨询功能。

### 1.2 项目目标
- **品牌展示**：建立专业、可信的中医诊所品牌形象，展现传统中医文化的价值
- **服务介绍**：清晰展示诊所提供的各类中医服务项目和诊疗范围
- **在线预约**：为患者提供便捷的在线预约功能，提高诊所运营效率
- **信息传达**：向患者传递诊所信息、联系方式、营业时间等关键信息
- **用户体验**：设计简洁大方、易于导航的网站界面，确保良好的用户体验

### 1.3 设计原则
- **简洁大方**：界面设计简洁明了，避免过多装饰，突出内容
- **专业性**：体现中医诊所的专业性和可信度
- **易用性**：导航清晰，功能易于使用，适配移动端
- **文化传承**：融入传统中医文化元素，但不喧宾夺主

## 二、系统用户与角色

### 2.1 主要用户群体

#### 2.1.1 患者/潜在患者
- **需求**：
  - 了解诊所服务项目和诊疗能力
  - 查看诊所信息和联系方式
  - 在线预约就诊时间
  - 进行健康咨询
- **特点**：
  - 可能不熟悉中医服务
  - 需要清晰的信息展示和引导
  - 重视诊所的专业性和可信度

#### 2.1.2 诊所工作人员（管理员）
- **需求**：
  - 查看和管理预约信息
  - 更新服务内容和诊所信息
  - 查看患者咨询和留言
- **特点**：
  - 需要简单的后台管理功能
  - 通常为非技术人员

## 三、系统总体功能模块

### 3.1 网站前端模块

#### 3.1.1 首页（Home）
**功能描述**：
- 展示诊所品牌标识和核心价值主张
- 提供主要服务项目预览（3-4个核心服务）
- 快速导航到预约和详细信息页面
- Hero 区域展示诊所欢迎信息和行动按钮

**内容要素**：
- 诊所名称和 Logo
- 简短标语/使命陈述
- 核心服务预览卡片（针灸、中药、推拿等）
- 关于我们简要介绍
- 快速预约按钮

#### 3.1.2 关于我们（About）
**功能描述**：
- 介绍诊所的背景和使命
- 展示专业团队和医疗资质
- 体现诊所的专业性和可信度

**内容要素**：
- 诊所简介
- 使命和价值观
- 专业团队介绍
- 诊所历史和成就（可选）

#### 3.1.3 服务项目（Services）
**功能描述**：
- 详细展示诊所提供的所有中医服务项目
- 每个服务包含：服务名称、描述、主要功效、适用症状

**服务项目列表**：
1. **针灸治疗**：传统针灸疗法，缓解疼痛，调理身体
2. **中药调理**：个性化中药配方，调理体质，增强免疫力
3. **推拿按摩**：专业推拿手法，舒缓筋骨，放松身心
4. **拔罐疗法**：疏通经络，活血化瘀，缓解肌肉酸痛
5. **艾灸治疗**：温经散寒，补气养血，增强体质
6. **中医咨询**：健康评估，养生指导，生活方式建议

#### 3.1.4 预约就诊（Appointment）
**功能描述**：
- 提供在线预约表单
- 收集患者基本信息和预约需求
- 支持服务选择和日期偏好

**表单字段**：
- 姓名（必填）
- 联系电话（必填）
- 电子邮箱（可选）
- 预约服务（必填，下拉选择）
- 期望日期（可选）
- 备注信息（可选，如症状描述）

**交互要求**：
- 表单验证
- 提交后显示确认信息
- 提示患者也可电话预约

#### 3.1.5 联系方式（Contact）
**功能描述**：
- 展示诊所的详细联系信息
- 提供在线咨询表单
- 显示营业时间和地址

**内容要素**：
- 诊所地址
- 联系电话
- 电子邮箱
- 营业时间
- 在线咨询表单（姓名、邮箱、留言）

#### 3.1.6 导航组件（Navigation）
**功能描述**：
- 全局导航栏，固定在页面顶部
- 响应式设计，支持移动端菜单
- 当前页面高亮显示

**导航项**：
- 首页
- 关于我们
- 服务项目
- 预约就诊
- 联系方式

#### 3.1.7 页脚组件（Footer）
**功能描述**：
- 提供快速链接和联系信息
- 版权信息

**内容要素**：
- 诊所简介
- 快速链接
- 联系方式
- 版权声明

## 四、技术实现

### 4.1 技术栈
- **框架**：Next.js 14（App Router）
- **语言**：TypeScript
- **样式**：Tailwind CSS
- **字体**：Inter（Google Fonts）

### 4.2 项目结构
```
tcm-weissenberger-lu.ch/
├── src/
│   ├── app/
│   │   ├── layout.tsx          # 根布局
│   │   ├── page.tsx            # 首页
│   │   ├── globals.css         # 全局样式
│   │   ├── about/              # 关于我们
│   │   ├── services/           # 服务项目
│   │   ├── appointment/       # 预约就诊
│   │   └── contact/           # 联系方式
│   └── components/
│       ├── Nav.tsx             # 导航组件
│       └── Footer.tsx         # 页脚组件
├── doc/                        # 文档目录
│   └── PRD.md                 # 产品需求文档
├── package.json
├── tsconfig.json
├── next.config.js
├── tailwind.config.js
└── postcss.config.js
```

### 4.3 设计规范

#### 4.3.1 色彩方案
- **主色调**：红色系（体现中医传统，温暖专业）
  - Primary: #ef4444 / #dc2626 / #b91c1c
- **中性色**：灰色系
  - 文本：gray-900, gray-700, gray-600
  - 背景：white, gray-50, gray-100

#### 4.3.2 排版
- **标题**：粗体，层次分明（h1: 4xl, h2: 3xl, h3: 2xl）
- **正文**：清晰易读，适当行距
- **间距**：使用 Tailwind 标准间距单位

#### 4.3.3 响应式设计
- 移动端优先（Mobile First）
- 断点：sm (640px), md (768px), lg (1024px)
- 导航栏移动端使用汉堡菜单

## 七、注意事项

1. **内容准确性**：确保所有医疗相关信息准确、合规
2. **隐私保护**：患者信息需要符合数据保护法规
3. **可访问性**：遵循 Web 可访问性标准
4. **加载性能**：确保图片和资源优化加载
5. **备份机制**：定期备份网站数据和配置
