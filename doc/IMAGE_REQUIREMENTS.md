## 图片优化建议

1. **格式选择**:
   - Logo/图标: SVG (矢量图，可缩放)
   - 照片: JPG (WebP 用于现代浏览器优化)
   - 透明背景: PNG

2. **文件大小**:
   - Hero 图片: < 300KB
   - 服务图片: < 200KB
   - 图标: < 50KB

3. **响应式图片**:
   - 考虑为不同屏幕尺寸生成多个版本
   - 使用 Next.js Image 组件自动优化

4. **Alt 文本**:
   - 所有图片都需要有描述性的 alt 文本
   - 使用多语言支持

---

## 颜色参考

- **主色调**: Deep Green (#2d5016) 或 Teal (#0d9488)
- **辅助色**: Light Green (#f0fdf4), Soft Teal (#ccfbf1)
- **中性色**: Warm Whites, Soft Grays
- **强调色**: Primary-600 (根据 Tailwind 配置)

---

## 生成顺序建议

1. **优先**: Logo (logo-dark, logo-light, favicon)
2. **其次**: Hero 图片 (hero-home, hero-about)
3. **然后**: 服务项目图片 (6张)
4. **最后**: 其他装饰性图片

---

## 注意事项

1. 所有图片应体现"现代整合、科学导向"的定位
2. 避免过度"东方神秘感"，保持专业医疗形象
3. 符合瑞士本地化审美：简约、清爽、可信
4. 所有图片不应包含可识别的个人（除非是医师肖像）
5. 保持一致的视觉风格和色彩方案
